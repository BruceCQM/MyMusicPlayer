# MyMusicPlayer
基于Vue框架的仿网易云音乐播放器

# 项目主要功能
- 实现了4个主要功能模块：发现音乐、推荐歌单、最新音乐、最新MV，每天推荐不同的音乐内容
- 支持使用关键字搜索歌曲、歌单以及MV，并且能够进行在线播放
- 该音乐播放器支持页面的前进后退功能

# 项目主要技术
- 使用vue脚手架搭建项目的基本结构
- 使用vue-router管理多个组件之间的切换关系
- 使用axios网络请求库进行数据请求
- 使用Element-ui组件库协助进行UI设计，提高开发效率
